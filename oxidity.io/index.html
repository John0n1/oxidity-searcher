Operational note

The wallet backend exposes `/api/bootstrap`, `/api/portfolio`, `/api/send/prepare`, `/api/send/broadcast`, `/api/activity`, `/api/catalog`, and `/api/ai/chat`. The current Chrome artifact is a packaged extension build; once a Web Store listing exists, this page should point to that stable install target instead of the raw zip.
